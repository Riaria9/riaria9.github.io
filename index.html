ML systems project

KV cache compression study

Eviction x Quantization Interaction

Worked on an MLSys-style empirical report studying how KV-cache token eviction and value quantization interact under an equal-memory protocol. The project compared sequential eviction/quantization orderings, MiniKV-style baselines, H2O-style heavy hitters, and a joint scoring strategy across Llama and Yi model families.

  • Showed that equal-byte-budget comparisons change the interpretation of compression quality.
  • At an aggressive 8.39 MB cache budget on Llama-3.2-3B, joint scoring reached 8.62 PPL versus 43.00/48.17 for sequential orderings.
  • Used error decomposition to explain when eviction and quantization noise cancel or reinforce.
